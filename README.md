## Getting Started

### Prerequisites

- Go 1.18+
- Git

### Installation

Set Gemini API_KEY to `API_KEY=` in `.env` file

### Build

```bash
go build
```

### Usage

```bash
./gmn howto list all files in current directory
./gmn howto list git commit log containing word "add"
```
